Vous souhaitez diviser votre terrain sans intention de bâtir ?

Nous vous accompagnons dans la réalisation de votre projet en:

• Etudiant la faisabilité de l’opération.
• Analysant vos actes, les servitudes et les règles d’urbanisme.
• Etablissant le plan de division.

• Etablissant le document permettant d’obtenir les nouveaux numéros cadastraux.
• Bornant la nouvelle limite issue de la division.

Ces missions permettent de constituer le dossier que le notaire utilisera pour rédiger son acte dans le cadre de la mutation d’une ou plusieurs des parcelles créées.
Ces missions ne permettent pas de garantir les limites périmétriques de la propriété ni les superficies des parcelles.

Pour garantir les limites de propriétés et les superficies des parcelles (obligatoire dans le cadre de la vente d’un terrain à bâtir), seul le bornage amiable (Art. 646 du Code Civil) permet de le faire.
Dans le cas d’une division de terrain en vue de bâtir, un dossier plus complet devra être réalisé.
